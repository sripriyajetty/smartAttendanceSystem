# smartAttendanceSystem
The Smart Attendance System using Face Recognition successfully automates the attendance process by combining computer vision, machine learning, and hardware integration. The system uses Haar Cascade for detection and LBPH for recognition, supported by a 10-frame stability check and confidence threshold to ensure reliability. The Tkinter GUI simplifies operation, while Arduino hardware provides immediate physical feedback. Attendance is stored in structured, date-based CSV files, eliminating manual effort and proxy fraud. The project demonstrates that an effective, low-cost attendance solution can be built using Python, OpenCV, and basic hardware with clear pathways for future enhancement using deep learning and cloud technology
